# Minecraft Fabric Server no GitHub Codespaces

Tutorial completo para criação e configuração de um servidor **Minecraft Java 1.21.11 com Fabric**, utilizando:

* GitHub Codespaces
* Crafty Controller
* Fabric Loader
* Fabric API
* Playit.gg

## Sobre o projeto

Este projeto demonstra como criar, administrar e disponibilizar um servidor Minecraft com mods utilizando um ambiente Linux no GitHub Codespaces.

Durante o processo, são abordados conceitos de:

* Administração de servidores
* Linux e terminal
* Java
* Gerenciamento de arquivos
* Configuração de rede
* Túneis de conexão
* Análise de logs
* Instalação de dependências
* Backup e recuperação

## Aviso

O GitHub Codespaces é um ambiente de desenvolvimento temporário e não uma hospedagem permanente.

Quando o Codespace for encerrado, todos os serviços em execução, incluindo o servidor Minecraft, o Crafty Controller e o Playit.gg, também serão interrompidos.

## Tecnologias utilizadas

* Minecraft Java 1.21.11
* Java 21
* Fabric Loader
* Fabric API
* Crafty Controller
* Playit.gg
* GitHub Codespaces
* Ubuntu Linux

## Etapas do tutorial

1. Criação do repositório
2. Configuração do Codespaces
3. Instalação do Java
4. Instalação do Crafty Controller
5. Criação do servidor Fabric
6. Instalação de mods
7. Configuração do Playit.gg
8. Acesso externo ao servidor
